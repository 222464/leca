# Leca

## Introduction 

Leca stands for Learned Elementary Cellular Automata.

## Requirements

- OpenMP (this will likely already be installed on your system)
- pybind11
- cmake

## Installation

> pip install .

## Importing and Setup

The Leca module can be imported using:

```python
import leca
```
